## Color Legend

### Buttons
- **Menu Button**: Dark translucent (#000 70%) + Blue border (#4a90e2)
- **Rogue Mode Button**: Red gradient (#e94560 โ #ef476f)
- **Zoom Controls**: Dark translucent (#000 70%) + Blue border (#4a90e2)

### Hover States
- **All Buttons**: Light blue tint (#4a90e2 30%) on hover
- **NPCs**: Scale from 1.0x to 1.2x on hover

### Game Elements
- **Player**: Blue square (#4a90e2)
- **NPCs**: Green circles (#50c878)
- **Selected NPC**: Yellow tint (#ffff00)
- **Hovered NPC**: Scaled 1.2x

### Tooltips
- **Background**: Black 90% opacity
- **Border**: Blue (#4a90e2)
- **Text**: White

### Rogue Mode
- **Red Tint**: Red 10% opacity overlay
- **Audit Trail**: Dark background with red accent
- **Abuse Panel**: Matches game's dark theme

## Responsive Behavior

### Small Window (800x600)
- All UI elements visible
- Zoom controls may be close to panel edge
- Menu and buttons positioned absolutely

### Medium Window (1200x800)
- Comfortable spacing
- All elements well-separated
- Optimal viewing experience

### Large Window (1920x1080+)
- Game scales to fill space
- UI elements stay in corners
- Map more visible at default zoom

---

## Interaction Flow

### Normal Mode Flow
```
1. Load game
2. See map with NPCs
3. Hover NPC โ See name tooltip + scale effect
4. Click NPC โ Data panel appears
5. Click Menu โ See restart option
6. Use zoom controls โ Adjust view
7. Resize window โ Game adapts
```

### Rogue Mode Flow
```
1. Click "Enter Rogue Employee Mode"
2. See warning modal โ Accept
3. See scenario intro โ Begin
4. Red tint applied
5. Audit trail appears
6. Abuse panel appears
7. Click Menu โ See "Exit Rogue Mode" option
8. Exit โ Return to normal mode
```

## Keyboard Shortcuts (Future)

Currently, there are no keyboard shortcuts. Planned:
- `Esc` - Open/close menu
- `+` - Zoom in
- `-` - Zoom out
- `0` - Reset zoom
- `M` - Toggle menu
- `Tab` - Cycle through NPCs

## Touch Support (Future)

Currently optimized for mouse/pointer. Planned:
- Pinch to zoom
- Double-tap to reset zoom
- Long press for tooltip
- Swipe for menu
